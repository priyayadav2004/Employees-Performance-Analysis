📊 Employee Attrition & Performance Analysis
📌 Project Overview

This project focuses on analyzing employee data to identify the key factors affecting employee attrition and performance.
The goal is to help organizations improve employee retention and decision-making using data-driven insights.

🛠️ Tools & Technologies Used
Python (Pandas, NumPy, Matplotlib)
Jupyter Notebook
Power BI
Excel / CSV

📂 Project Structure
messy_data.csv → Raw dataset
cleaned_hr_attrition.csv → Cleaned dataset
Clean.ipynb → Data cleaning and analysis
Internship project.pbix → Power BI Dashboard
INDUSTRY PROJECT REPORT.pdf → Project report

🔍 Data Cleaning Steps
Handled missing values
Removed duplicates
Data type conversion
Feature transformation
📊 Analysis & Visualization

Created histograms for numeric features
Identified patterns in employee attrition
Built interactive dashboard in Power BI
📈 Key Insights
Attrition is higher in certain departments
Salary and experience significantly impact retention
Work-life balance plays an important role
🚀 Conclusion
This project demonstrates how data analysis can be used to uncover meaningful insights and support better HR decisions.
